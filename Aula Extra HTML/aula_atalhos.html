Definições:
            Html: Hypertext Markup Language (
                Linguagem de Marcação de Hipertexto)

            Https: Hypertext Transfer Protocol Secure (
                Protocolo de Transferência de Hipertexto Seguro)

            CSS: Cascading Style Sheets (Folha de Estilo em Cascata)

            IDE: Integrated Development Environment (Ambiente de Desenvolvimento Integrado)
            Atalhos VS Code:
            !: Criar o head do Html 5
            Alt + Shift + F: Identar;
            Alt + Z: Deixar o Código na mesma linha;
            Alt + Shift + A: Criar Comentário;
            Control + ;: Comentar linha;
            Control + / : Comentar linha inteira;
            Control + H: Replace;
            Control + Shift + Alt + Seta: Selecionar diversas linhas simultaneamente;
            Control + Seta: Descer a página (scroll do mouse);
            Alt + Seta: Mover linha do código;
            Control + Space: Sugestões de Preenchimento do Código

            Ctrl + Shift + I ou F12 = Inspecionar Página
